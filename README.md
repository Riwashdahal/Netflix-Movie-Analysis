# Netflix-Movie-Analysis
# 🎬 Netflix Movie Data Analysis

## 📌 Project Overview

This project analyzes a Netflix movies dataset to uncover insights about content trends, user preferences, and key factors influencing movie popularity. The goal is to demonstrate practical data analysis skills using Python, focusing on data cleaning, exploration, visualization, and storytelling.

---

## 🎯 Objectives

* Understand the distribution of movies across genres and years
* Analyze ratings and popularity trends
* Identify top-performing genres, directors, and actors
* Explore relationships between features such as release year, duration, and ratings
* Generate actionable insights from the dataset

---

## 🛠️ Tools & Technologies

* **Python**
* **Pandas** – Data manipulation and cleaning
* **NumPy** – Numerical computations
* **Matplotlib & Seaborn** – Data visualization
* **Jupyter Notebook** – Analysis environment

---

## 📂 Dataset

* Source: Netflix Movies Dataset (CSV format)
* Contains information such as:

  * Title
  * Genre
  * Release Year
  * Duration
  * Rating
  * Director / Cast (if available)

---

## 🔍 Key Analysis Steps

1. **Data Loading & Inspection**

   * Loaded dataset using Pandas
   * Checked structure, data types, and missing values

2. **Data Cleaning**

   * Handled missing/null values
   * Removed duplicates
   * Standardized column formats

3. **Exploratory Data Analysis (EDA)**

   * Distribution of movies by year
   * Genre frequency analysis
   * Ratings analysis
   * Duration trends

4. **Data Visualization**

   * Bar charts for genre distribution
   * Line plots for yearly trends
   * Histograms for rating distribution
   * Heatmaps for correlation analysis

---

## 📊 Key Insights

* Certain genres dominate Netflix content (e.g., Drama, Comedy)
* Rapid increase in content production in recent years
* Ratings distribution shows clustering around mid-to-high values
* Duration varies significantly across genres

## 🚀 How to Run the Project

1. Clone the repository:

   ```bash
   git clone <your-repo-link>
   ```
2. Navigate to the project folder:

   ```bash
   cd netflix-movie-analysis
   ```
3. Install required libraries:

   ```bash
   pip install pandas numpy matplotlib seaborn
   ```
4. Open the Jupyter Notebook:

   ```bash
   jupyter notebook
   ```
